Page scanning (AI vision)

Used by "Scan a page" in the lesson/quiz editor. Configure one or both providers — the primary is tried first and the other is used as automatic fallback.

Primary provider
Local — Ollama

Install a vision model: ollama pull llava or ollama pull llava-llama3

Ollama Cloud (free tier)

Runs through your local Ollama but offloads compute to Ollama's cloud. Get a free API key at ollama.com → Account → API Keys, then pull the model: ollama pull gemma4:cloud. Enter the base name only (e.g. gemma4) — :cloud is added automatically.
